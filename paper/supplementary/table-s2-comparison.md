# Table S2. Landscape comparison matrix

The finished table will compare BioMCP against adjacent systems and artifacts that overlap with biomedical retrieval, tool use, or structured interfaces.

## Expected columns

| Column | Meaning |
| --- | --- |
| `System` | Name of the compared system or package. |
| `Paper / artifact` | Citation handle or artifact name for the compared system. |
| `Year / venue` | Publication year or release venue. |
| `Source coverage signal` | Brief summary of how source breadth is demonstrated. |
| `Open source` | Whether the compared system is publicly available. |
| `CLI` | Whether the system exposes a command-line interface. |
| `MCP` | Whether the system exposes an MCP-compatible surface. |
| `Install surface` | Primary installation path or packaging surface. |
| `Citation count` | Citation or usage count noted during review. |
| `BioMCP-relevant note` | Reviewer note explaining why the system belongs in the comparison set. |
