# Table S4. Source citations

The finished table will list the canonical citations used for the external sources referenced throughout the BioMCP paper package.

## Expected columns

| Column | Meaning |
| --- | --- |
| `Source` | Source or database name. |
| `Citation` | Preferred citation text for the source. |
| `DOI` | DOI for the cited source when available. |
| `PMID` | PMID for the cited source when applicable. |
| `Status` | Tracking note for citation completeness or verification. |
