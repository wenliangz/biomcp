# Table S1. Leaf-style source enumeration

The finished table will enumerate the leaf-style biomedical sources BioMCP can reach and how those source signals surface in the product.

## Expected columns

| Column | Meaning |
| --- | --- |
| `#` | Reviewer-facing row number for the source entry. |
| `Source` | Canonical source or database name. |
| `How BioMCP reaches it` | Command surface or integration path used to reach the source. |
| `Entity surfaces` | Entity cards or search families where the source appears. |
| `Verification note` | Short note about how the source link or citation is verified. |
